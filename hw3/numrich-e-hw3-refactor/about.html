About Game
This is a game I made back in IGME-235 that uses PokeAPI as a base to play a game inspired by the Pokémon anime's game under the same name. This particular rendition is what I did back then but reworked with a new UI and code that uses Bulma and TypeScript. The code is also reword to utilize modules and includes a navigation bar for easy traversal between this page and the game. The code is also all bundled in ES5.
